# React Country Finder

A simple React web application that allows users to search for a country and view its details such as flag, capital, population, and region.

## Features
- Search any country
- View country flag and information
- Fetch data using the REST Countries API

## Tech Stack
- React
- Vite
- JavaScript
- REST Countries API
